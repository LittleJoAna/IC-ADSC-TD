1. Операционные системы, которые можно использовать при обучении
	1.1. Linux-системы, например Ubuntu
	1.2. Microsoft Windows
	1.3. Mac OS
2. Среда разработки, от простого к сложному
	1.1. Блокнот, консоль и MinGW(варианты CygWin, mingw-w64(простая установка))
	1.2. CodeBlocks(простой вариант, качать с mingw в названии инсталятора - CodeBlocks-17.12mingw-setup.exe)
	1.3. Eclipse и MinGW(варианты CygWin, mingw-w64(простая установка))
	1.4. QTCreator(исходный инсталятор 19МБ, в окне установки нужно выбрать версию для установки и
		желательно только вариант MinGW, иначе объём установки под 40ГБ.
		Скачивание конкретной версии занимает для MinGW занимает минут 10 и ещё пару минут установка.)
3. Установка ПО
	3.1. Установка для Microsoft Windows
		3.1.1. Установка mingw-w64. Пример "Hello World" в блокноте.
		3.1.2. Установка CodeBlocks. Пример "Hello World".
		3.1.3. Установка Eclipse+mingw-w64. Пример "Hello World".
		3.1.4. Установка QTCreator. Пример "Hello World".
	3.2. Установка для Ubuntu
		3.2.1. Установка gcc. Пример "Hello World" в тамошнем блокноте(на выбор).
		3.2.2. Установка CodeBlocks+gcc. Пример "Hello World".
		3.2.3. Установка Eclipse+gcc. Пример "Hello World".
		3.2.4. Установка QTCreator. Пример "Hello World".
	3.3. Установка для MacOS
		3.3.1. Установка gcc. Пример "Hello World" в тамошнем блокноте(на выбор).
		3.3.2. Установка CodeBlocks+gcc. Пример "Hello World".
		3.3.3. Установка Eclipse+gcc. Пример "Hello World".
		3.3.4. Установка QTCreator. Пример "Hello World".
